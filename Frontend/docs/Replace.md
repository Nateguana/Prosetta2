# Rep - List Replace

Sets the value at a specified index in the list, replacing whatever is currently there.

## Arguments

```list (list), index (number), value (any)```

## Example
<editor :code="`
was list lis 1 2 3..
rep list 1 1.
pri list.
`"
:code-wordier="`
Was list listing 1, 2, and 3?
Replace the list 1 to 1.
Print the list.
`" output-method='console'></editor>
