# Tur - Rotate

Turns the turtle by a specified number of degrees. All shapes drawn will use this rotation value.

## Arguments

```degrees (number)```

## Example
<editor :code='`
tur 45.
pen 15.
rec 100.
`' 
:code-wordier="`
Turn 45 degrees.
Set the pen to 15.
Then wreck 100 homes!
`"
output-method='canvas'></editor>
