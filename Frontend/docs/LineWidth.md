# Pen - Line Width

Sets the stroke width to a specified number. Also closes the current shape.

## Arguments

```size (number)```

## Example

<editor :code="`
Line Width Example
by Milo Jacobs and John Graphton\n
pen 15.
lin 50 50.
`"
